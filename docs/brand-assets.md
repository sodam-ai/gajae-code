# Brand assets

Gajae-Code uses the current GJC character and hero images in `assets/` for README and documentation surfaces.

| Asset | Purpose |
| --- | --- |
| [`assets/hero.png`](../assets/hero.png) | Wide README/docs hero image for Gajae-Code. |
| [`assets/character.png`](../assets/character.png) | Standalone Gajae-Code character mascot. |
| [`assets/tool-image-fixture.webp`](../assets/tool-image-fixture.webp) | Minimal WebP fixture for terminal image rendering tests. Not a product brand asset. |

The old legacy demo artwork has been removed from the active asset set; new public surfaces should reference the Gajae-Code assets above.
